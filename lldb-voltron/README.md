* `.lldbinit-lldb`: lldb init script. Based on voltron.
* `voltron.yml`: tmuxinator config file for using voltron + lldb.
    - file path: `<HOME>/.config/tmuxinator/`
    - `tmuxinator start|edit|delete voltron`: launch|edit|delelte
* `voltron.alias`: rc file for voltron.
* `telescope.py`: voltron plugin for smart-dereferencing pointer.
    - file path: `<HOME>/.voltron/plugins/`
